# 2xgen

A modern React-based website showcasing 2xGen's services and projects.

## Features

- Modern React frontend with Vite
- Responsive design with Tailwind CSS
- Project showcase and management
- Contact forms and insights blog
- Admin panel for project management
- Firebase integration for data persistence

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Firebase
- React Router

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser and navigate to the local development URL

## Project Structure

- `src/components/` - React components
- `src/pages/` - Page components
- `src/data/` - Static data and JSON files
- `src/services/` - API and service functions
- `public/` - Static assets

## Deployment

This project is configured for deployment on Vercel with automatic builds from the main branch.
